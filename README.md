チャットボードと言っていますが、シンプルなリアルタイムチャットアプリだと思っていただいて構いません。
「ボード」をユーザーが作成し、そのボード内でチャットができる仕様になっています。
非同期処理に加え、他ユーザーの更新情報も非同期（リアルタイム）に反映されます。
